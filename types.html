cloudify.nodes.FileSystem¶

Derived from: cloudify.nodes.Root

Properties:

fs_type

required

The type of the File System. Supported types are [ext2, ext3, ext4, fat, ntfs, swap]

use_external_resource

default: False

Enables the use of already formatted volumes.

fs_mount_path

required

The path of the mount point.

partition_type

default: 83

The partition type. 83 is a Linux Native Partition.
